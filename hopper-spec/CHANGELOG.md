# Changelog

This file tracks changes to the Hopper language specification. Entries are added when a spec version is released. The current working draft is unreleased.

---

## Format

Each release entry follows this format:

```
## [version] — YYYY-MM-DD

### Breaking Changes
- Description of any change that requires updating conforming programs or implementations.

### Added
- New features, syntax, or normative requirements added in this version.

### Changed
- Modifications to existing normative requirements.

### Fixed
- Corrections to specification text that did not change intended behavior.

### Removed
- Features or requirements removed in this version.
```
